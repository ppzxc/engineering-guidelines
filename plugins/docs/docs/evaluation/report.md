## 평가 실행 방법

각 테스트케이스를 Claude Code 세션에서 **입력 상황**에 따라 스킬을 호출하고,
**검증 포인트**를 기준으로 COVERED / PARTIAL / MISSING을 판정한다.

| 판정 | 기준 |
|------|------|
| COVERED | 검증 포인트 100% 충족 |
| PARTIAL | 검증 포인트 일부만 충족 |
| MISSING | 검증 포인트 미충족 또는 동작 없음 |
