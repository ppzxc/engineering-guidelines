# engineering-guidelines

A personal Claude Code marketplace containing engineering guidelines and best practices for software development.

## Plugins

| Plugin | Description |
|--------|-------------|
| [restful-api-guidelines](./plugins/restful-api-guidelines) | RESTful API design guidelines — URL structure, HTTP methods, status codes, JSON format, error responses, versioning, and headers |

## Install

```bash
claude plugin marketplace add https://github.com/ppzxc/engineering-guidelines.git
```
